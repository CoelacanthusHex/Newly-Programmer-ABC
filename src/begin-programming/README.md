# 开始编程

## 章节说明

**行内的代码**会用等宽字体表明，它通常像下面这样：`inline code`。

像下面这样的段落叫做“**代码块**”，它通常包含**多行代码**，一般用来向读者展示终端中执行的命令，或者若干代码语句：

可能是在终端中的输入的命令：

```console
$ cat 1.txt 2.txt > 3.txt
```

一些代码语句：

```cpp
int i = 1, j = 2;
double a = 3.14159;
```

像下面这个代码块展示了一个完整的代码文件的内容，这通常会通过在**代码块首行**以注释的方式标出**文件名**来表示：

```cpp
// hello-world.cpp

#include <iostream>

int main() {
    std::cout << "Hello!";
}
```
